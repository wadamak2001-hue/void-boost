# **App Name**: VOID BOOST

## Core Features:

- Intelligent Game Analyzer: Uses an AI performance advisor tool to analyze hardware usage and suggest the optimal performance profile for specific game titles.
- One-Tap Nebula Boost: Trigger a rapid system cleanup sequence with high-energy neon animations and tactile vibration feedback.
- Live Stats Holo-Monitor: Display real-time FPS simulations, CPU temperature, and memory utilization via a minimalist neon dashboard.
- Glassmorphic Game Library: A centralized, stylish game launcher featuring high-contrast neon card layouts for all your favorite titles.
- Smart Do-Not-Disturb Toggle: Quick-action toggle to silence notifications and optimize system focus specifically for high-intensity gaming sessions.
- Performance Profile Customization: A refined UI suite to toggle specialized modes like 'Battery Saver' or 'Turbo Zero-Lag'.

## Style Guidelines:

- The palette centers on 'Void Neon' aesthetics: A Primary Neon Blue (#00BFFF) for energy, complemented by a deep Midnight Indigo background (#0A0C12), and an analogous Purple Accent (#7462FF).
- We recommend a pairing: 'Poppins' for striking techy headlines and 'Inter' for clear, legible performance metrics. Note: currently only Google Fonts are supported.
- Thinline, glowing icons with rounded terminals to maintain the futuristic, high-tech performance aesthetic.
- Minimalist 'Glassmorphism' layout featuring semi-transparent surfaces, soft blurs, and wide spacing for a clean, premium experience.
- Smooth 60FPS transitions with 'Glowing pulse' effects on the Boost button and subtle shake-on-tap haptic simulations.
